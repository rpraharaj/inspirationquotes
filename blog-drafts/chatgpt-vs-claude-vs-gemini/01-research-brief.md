## 3. Current AI Models (as of January 9, 2026)

### OpenAI (ChatGPT)
- **Latest Model:** GPT-5.2 (December 2025)
- **Key Variants:** GPT-5.2 Pro, GPT-5 mini, GPT-5.1 Thinking, GPT-5.1 Instant
- **Context Window:** 128K tokens (GPT-5.2), 400K tokens (GPT-5.2 extended)
- **Market Share:** 68% (down from 87.2% in 2025)
- **Weekly Users:** 800 million
- **Pricing:** Free tier, Plus $20/mo, Pro $200/mo

**Strengths:**
- Best for creative writing
- Strong coding (95% accuracy)
- Versatile all-purpose assistant
- Large ecosystem (Custom GPTs, plugins)
- Memory and Projects features

### Anthropic (Claude)
- **Latest Model:** Claude Opus 4.5 (November 2025), Claude Sonnet 4.5 (September 2025)
- **Context Window:** 200K tokens (expandable to 1 million)
- **Key Feature:** 80.9% on SWE-bench, 93.7% coding accuracy
- **Pricing:** Pro $20/mo, Max tiers $100-200/mo

**Strengths:**
- Superior coding performance
- Largest effective context window for long docs
- Safest/most reliable (fewer hallucinations)
- Deep research and analysis
- Natural writing tone
- Claude Code for developers

### Google (Gemini)
- **Latest Model:** Gemini 3 Pro, Gemini 3 Flash
- **Context Window:** 1 million to 2 million tokens
- **Speed:** 372 tokens/second (Gemini 2.5 Flash fastest)
- **Market Share:** 18.2% (up from 5.4%)
- **Pricing:** Free tier, Premium ~$20/mo, Ultra $249.99/mo

**Strengths:**
- Native multimodal (text, image, video, audio)
- Massive context window
- Google ecosystem integration
- Real-time web search
- Video analysis capabilities
- Fast response speeds
