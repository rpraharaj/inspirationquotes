## 2. Current AI Coding Tools (January 2026)

### GitHub Copilot
- Powered by GPT-4o (default model)
- Market share: 60%+ in 2026
- Pricing: $10/month Pro, $19/month Business
- Key feature: Deep GitHub ecosystem integration
- Context: Primarily file-focused autocomplete

### Cursor
- AI-first code editor (VS Code fork)
- Supports Claude 3.7, GPT-4o, custom models
- Pricing: Free tier, $20/month Pro, ~$200/month Enterprise
- Key feature: Multi-file editing, Composer mode
- Context: Full codebase indexing

### Sourcegraph Cody
- Code intelligence platform with AI
- Enterprise-grade code understanding
- Pricing: Free starter, $49/user/month Enterprise
- Key feature: Large codebase comprehension
- Context: Cross-repository understanding

## 4. Key Differentiators

### GitHub Copilot Strengths
- Industry standard, most developers familiar
- Seamless GitHub integration (PRs, Issues, Actions)
- Wide IDE support
- Free for students
- GPT-5 powered (latest)

### Cursor Strengths
- Multi-file refactoring capabilities
- Deep codebase understanding
- AI-native editor design
- Model flexibility (choose LLM)
- Fast inline suggestions

### Cody Strengths
- Enterprise-scale code understanding
- Cross-repository context
- On-premise deployment option
- Natural language code search
- Legacy codebase navigation
