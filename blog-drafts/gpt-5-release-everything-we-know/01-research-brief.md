### Key Features
1. **Unified Architecture** - Single system integrating memory, reasoning, vision, and task execution
2. **Multimodal Natively** - Text, images, audio, video processed together
3. **Agentic Functionality** - Can set up desktop, browse web, execute multi-step tasks autonomously
4. **Adjustable Thinking Levels** - Auto, Fast, Thinking modes for different task complexities
5. **Reduced Hallucinations** - 45% fewer factual errors than GPT-4o
6. **Enhanced Coding** - Strongest coding model to date, creates complete apps from prompts
7. **ChatGPT Voice** - Replaces Advanced Voice Mode, more natural conversations

### Launch Issues & Controversies
1. **Initial Performance Complaints** - Users reported GPT-5 was less capable than GPT-4o initially
2. **Robotic Responses** - Shorter, less engaging, lacked personality of earlier models
3. **Model Choice Removed** - Users couldn't select GPT-4o without warning
4. **Rate Limits for Plus** - Initially 200 messages/week for Thinking mode (increased to 3,000)
5. **Autoswitcher Problems** - Model routing system caused incoherent answers
6. **GPT-5.2 Rush** - Concerns about rushed release due to competition from Claude and Gemini

### OpenAI's Responses
- Reinstated GPT-4o access for Plus users
- Increased Thinking mode limits from 200 to 3,000 messages/week
- Sam Altman admitted: "We totally screwed up some things"
- Added thinking modes (Auto, Fast, Thinking) for user control
- Promised future personalization features for response style
