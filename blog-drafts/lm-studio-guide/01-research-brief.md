## 3. Key Features to Cover

### LM Studio Capabilities (2025-2026)
- Intuitive GUI for model discovery and download
- One-click model downloads from Hugging Face
- Built-in chat interface
- OpenAI-compatible local API server (localhost:1234)
- GPU acceleration (NVIDIA CUDA, Apple MLX)
- CPU-only inference option
- Document chat (RAG) - chat with your files
- Python and TypeScript SDKs (March 2025)
- Speculative Decoding for faster inference
- Tool Calling API (beta)
- Multi-GPU controls

### Hardware Requirements
- Minimum 8GB VRAM for 7B models (Q4)
- 16GB VRAM recommended for quality
- CPU-only possible but slower
- Supports Apple Silicon, NVIDIA, AMD (Vulkan)
