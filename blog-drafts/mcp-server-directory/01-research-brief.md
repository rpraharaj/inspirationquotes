## Topic Research

### What is MCP?
Model Context Protocol (MCP) is an open standard created by Anthropic in November 2024 that standardizes how AI systems (like Claude) connect to external tools, data sources, and services. It uses a client-server architecture:
- **MCP Host:** AI application (Claude Desktop, IDEs)
- **MCP Client:** Translator within host
- **MCP Server:** External service providing tools/data
