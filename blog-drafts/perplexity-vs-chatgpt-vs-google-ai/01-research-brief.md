## 5. Current AI Research Tool Versions (as of January 2026)

### Perplexity AI
- **Current Status:** Leading AI-powered answer engine
- **Market Share:** 3.1% of AI search engines
- **Key Features:**
  - Real-time web search with live data
  - Source citations for every answer
  - "Spaces" for organizing research projects
  - Pro Searches with multi-step reasoning
  - File analysis (PDF, CSV, images, audio, video)
  - Access to GPT-4o, Claude 3.5, Gemini 2.5 Pro, Sonar
  - Labs for reports, spreadsheets, web apps
  - Average response time: 1.2 seconds
- **Pricing:**
  - Free: Limited queries, basic features
  - Pro: $20/month ($200/year) - 300+ Pro searches daily
  - Max: $200/month ($2,000/year) - Unlimited advanced models
  - Enterprise Pro: $40/seat/month
  - Enterprise Max: $325/seat/month

### ChatGPT with Web Browsing
- **Current Version:** GPT-5o with integrated web search
- **Key Features:**
  - "Deep Research Assistant" mode for automated reports
  - Multimodal processing (text, images, audio, video)
  - Average 10.42 links per response
  - Memory for personalized interactions
  - GPT Store for custom research GPTs
  - Agentic capabilities for complex tasks
- **Pricing:**
  - Free: Basic chat, limited web browsing
  - Plus: $20/month - Enhanced web access
  - Pro: Custom pricing - Advanced features
  - Enterprise: Custom pricing

### Google AI Search (Gemini + AI Overviews)
- **Current Version:** Gemini 3 Pro powering AI Overviews
- **Key Features:**
  - "Answer-first" AI Overviews in search results
  - Deep integration with Google ecosystem
  - Multimodal search (text, voice, visual)
  - Personalized results based on search history
  - Integration with Gmail, Docs, etc.
  - Dominant market position
- **Pricing:**
  - Free: AI Overviews in standard search
  - Google AI Pro: Enhanced features across services
  - Workspace: Business features
