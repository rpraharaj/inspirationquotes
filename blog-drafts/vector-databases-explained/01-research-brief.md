## Key Concepts
- Embeddings: Numerical representations of data
- Similarity search: Finding semantically similar items
- Approximate Nearest Neighbor (ANN) algorithms
- Cosine similarity, Euclidean distance

## Popular Vector Databases
1. **Pinecone** - Fully managed, serverless, production-ready
2. **Weaviate** - Open-source, hybrid search, GraphQL API
3. **Chroma** - Open-source, developer-friendly, great for prototyping
4. **Milvus** - Open-source, highly scalable
5. **Qdrant** - Open-source, Rust-based
6. **PostgreSQL + pgvector** - Extension for existing Postgres

## Use Cases
- Semantic search
- RAG (Retrieval-Augmented Generation)
- Recommendation systems
- Image/video similarity
- Anomaly detection
