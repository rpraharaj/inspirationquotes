# Changelog

All notable changes to the Claude Agents Library will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.2.0] - 2026-01-16

### Added
- **Agent Selection Guide** - Comprehensive guide to finding the right agents by role, task, or team size
  - 18 role-based recommendations
  - 20 common task mappings with time savings
  - 5 team structure guides (Solo to Enterprise)
  - Alphabetical quick-find index with all 34 agents
  - Decision tree for first-time users

- **MCP Integration Guide** - Complete implementation guide for Model Context Protocol
  - 7 production-ready MCP + Agent workflow patterns
  - Step-by-step implementation instructions with code
  - Security best practices for MCP servers
  - Multi-environment setup examples
  - Troubleshooting guide
  - Time savings metrics for each pattern

- **Model Selection Guide** - Comprehensive model selection and cost optimization
  - 30+ task-specific model recommendations
  - Detailed cost breakdowns per task type
  - Extended Thinking usage guidelines
  - 4 cost optimization strategies with examples
  - Real-world monthly cost scenarios
  - Decision tree for model selection
  - Cost optimization checklist

- **Enhanced README**
  - Quick agent finder tables (by role, task, team)
  - MCP integration highlights with time savings
  - Cost optimization summary (82% savings example)
  - Version history section
  - Links to new comprehensive guides

### Changed
- Updated README structure for better scannability
- Enhanced MCP section with concrete examples
- Improved agent categorization with use case examples
- Added time savings percentages throughout

### Improved
- Better discoverability of agents through multiple selection methods
- Practical, copy-paste ready workflows instead of theoretical descriptions
- Quantified value (time savings %, cost reductions)
- Professional presentation with comprehensive documentation

## [1.1.0] - 2026-01-13

### Added

- Claude-specific best practices
- EU AI Act compliance updates to Legal Compliance Checker agent

### Changed

- Rebranded Twitter references to X/Twitter 

### Improved
- Enhanced agent documentation with real-world examples
- Better cross-referencing between related agents

## [1.0.0] - 2025-11-25

### Added
- Initial release with 34 production-ready agents
- 7 professional categories:
  - Engineering (6 agents)
  - Product (3 agents)  
  - Marketing (7 agents)
  - Design (5 agents)
  - Project Management (3 agents)
  - Studio Operations (5 agents)
  - Testing (5 agents)
- Comprehensive agent structure:
  - Purpose, Responsibilities, Skills
  - Communication Style, Example Prompts
  - Related Agents mapping
- MIT License for open use
- Contribution guidelines
- Basic usage documentation
