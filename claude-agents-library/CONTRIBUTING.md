# Contributing to Claude Agents Library

Thank you for your interest in contributing! This library grows stronger with community input.

## How to Contribute

### Adding a New Agent

1. **Fork the repository**
2. **Create your agent file** in the appropriate category folder
3. **Follow the agent template** (see below)
4. **Submit a pull request** with a clear description

- Add new example prompts
- Update tools and technologies

## Pull Request Process

1. Ensure your agent follows the template
2. Update the README if adding a new agent
3. Include a brief description of changes
4. Reference any related issues

## Code of Conduct

Be respectful, constructive, and helpful. We're all learning together.
