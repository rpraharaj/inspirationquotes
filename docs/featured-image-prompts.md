## Master Template

Use this template structure for all AI image generation. Replace placeholders with actual values.

```
Create a featured image for a tech blog (1200x630 pixels):

TITLE: "[POST TITLE]"

DESIGN REQUIREMENTS:
1. BACKGROUND: Clean gradient from pure black (#000000) to dark gray (#171717)
2. TITLE: Bold white (#FFFFFF) text, centered horizontally and vertically
3. SAFE ZONE: Keep ALL text in the CENTER 50% of the image height
   - Top 25% = background only (will be cropped on some platforms)
   - Middle 50% = title and main content
   - Bottom 25% = background only (will be cropped on some platforms)
4. ICONS: Add subtle [CATEGORY] themed outline icons in light gray (#A3A3A3)
   - Place icons decoratively around the title
   - Keep icons subtle and non-distracting
5. WATERMARK: Add "www.aiagentskit.com" in small text (16px)
   - Position: Bottom right corner
   - Color: Subtle gray (#525252)
   - Font: Clean sans-serif
6. STYLE: Minimalist, professional, tech-focused
7. COLORS: Black, white, and grayscale ONLY - no other colors

The image should look professional and suitable for a technology blog about AI.
```

## Tips for Best Results

### Do's ✅

1. **Be specific about dimensions** - Always specify 1200x630 pixels
2. **Emphasize the safe zone** - Remind about middle 50% for content
3. **Request the watermark explicitly** - Include exact text and position
4. **Mention grayscale only** - Prevent color bleeding
5. **Keep title short** - Under 60 characters works best
6. **Request high contrast** - White on black for readability

### Don'ts ❌

1. **Don't use complex backgrounds** - Keep gradients simple
2. **Don't request colored icons** - Stay black & white
3. **Don't skip the watermark** - Always include it
4. **Don't put text near edges** - Social platforms crop aggressively
5. **Don't use thin fonts** - Bold, heavy fonts work better

### Title Formatting Tips

| Title Length | Font Size | Lines |
|--------------|-----------|-------|
| < 30 chars | Large (72px) | 1 line |
| 30-50 chars | Medium (64px) | 1-2 lines |
| 50-70 chars | Small (56px) | 2 lines |
| > 70 chars | Very small (48px) | 2-3 lines |

## Processing AI-Generated Images

After generating with AI, process the image to ensure compliance:

```bash
# Basic processing (crop + convert)
node scripts/process-image.cjs ai-generated.png public/images/featured/my-post.webp

# With watermark (if AI forgot it)
node scripts/process-image.cjs ai-generated.png public/images/featured/my-post.webp --watermark

# With reduced quality (if file too large)
node scripts/process-image.cjs ai-generated.png public/images/featured/my-post.webp --quality 70
```

### Verification Checklist

After processing:

- [ ] Dimensions: 1200×630
- [ ] Format: WebP
- [ ] Size: <100KB
- [ ] Watermark: Visible
- [ ] Title: Readable
- [ ] Safe zone: Content centered
